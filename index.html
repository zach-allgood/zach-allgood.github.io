About Me

The Beginning

I started programming when I was 7. I started off in visual basic but quickly moved to C++. I started developing web sites when I was around 12 years old. I experimented with a lot of languages up until I was about 16. Some of the the experiments included game development, desktop application development, and Minecraft mods. At 16, I really started focusing on web development and how to improve as a programmer overall. I got an internship as a a software engineer intern at Classy Llama in October of 2015 and started my career.

School

I took my first programming class when I was a sophomore at Kickapoo High School. I learned RobotC using hands-on experience with robots. In my junior year, I took a web design class and learned the basics of web development. In that year, I also took Intro to Computer Programming and Advanced Computer Programming, which covered Java and the basic thought patterns and concepts that a programmer should be comfortable with. In my senior year, I really focused on learning the trade. I took multiple different classes including Intro to Computer Science, Web Site Development I, Problem Solving and Programming Concepts, C# Programming I, and Systems Analysis and Design. All of the programming classes that I had taken my senior year except Intro to Computer Science were taught at Ozark Technical Community College (OTC).

The Present

Currently I am a Jr. Software Engineer at Classy Llama in Springfield, MO. Although my main focus is web development, I haven't completely dropped everything else. I do still work with C#, Java, and Python every now and then when I feel like refreshing myself in the languages. I enjoy the backend side of programming rather than the UI or frontend, however I do still enjoy them as well. I'm a very logical person and I love math, so it makes sense that I prefer working with algorithms and complex problem solving.
